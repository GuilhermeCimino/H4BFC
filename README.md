Site de Futebol – H4BFC

Este projeto é um sistema completo para gestão de funcionários, login autenticado por token e compra de ingressos para partidas de futebol.
Ele foi desenvolvido para rodar localmente utilizando XAMPP como servidor e Java 17 como ambiente principal.

- Funcionalidades
- Cadastro de Funcionários

Registro de novos funcionários

Edição e remoção de cadastros

Armazenamento seguro no banco de dados

- Login com Token

Autenticação via JWT

Sessões seguras

Acesso restrito a usuários autorizados

- Compra de Ingressos

Seleção de partidas

Escolha de categorias e quantidade de ingressos

Processamento e confirmação da compra

- Tecnologias Utilizadas

Java 17 – Backend

XAMPP – Servidor local

MySQL – Banco de dados

HTML / CSS / JavaScript – Interface web

API com Token – Segurança e autenticação

- Pré-requisitos

Antes de rodar o projeto, certifique-se de ter instalado:

- Java 17

Este projeto utiliza a versão 17 do JDK.

- XAMPP
